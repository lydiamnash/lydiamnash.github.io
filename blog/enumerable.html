Cycle, an Enumerable Method

1/11/2016

The method Cycle for Ruby Enumerables allows a user to run a code block multiple times or infinitely. Here is an example below of how the method can be used:

array = [0, 5, 10, 15, 20, 25] array.cycle(2) { |x| puts x + 1 } => 1, 6, 11, 16, 21, 26, 1, 6, 11, 16, 21, 26 array.cycle { |x| puts x + 1 } => 1, 6, 11, 16, 21, 26, 1, 6, .... infintely

As you can see above, if the user does not specify the number of cycles he/she wishes the code block to run, it will run infinitely.
