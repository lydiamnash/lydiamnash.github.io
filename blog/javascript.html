Javascript Loops

1/25/2016

For the past several weeks of DBC, we have been learning Ruby, but this last week we transitioned to a week of Javascript. This blog post will discuss some of the similarities and differences betweeen loops in JS and Ruby.

Here is an example of a loop in Javascript:

var my_array = [0,2,4,6];
for (i=0; i < array.length; i++) {
my_array[i] += 2
};

Here is an example of a loop in Ruby:

my_array = [0,2,4,6]
my_array.each do |item|
item += 2
end
