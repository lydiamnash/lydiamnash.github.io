SQL Cheat Sheet

Creating, Deleting, and Modifying tables

Create a Table

CREATE [table_name] ([column1_name] [column1_data_type],[column2_name] [column2_data_type],...)

Delete a Table

DROP TABLE [table_name]

Modify a Table (Add or delete column)

ALTER TABLE [table_name] ADD [column__name] [column_data_type] ALTER TABLE [table_name] DROP [column__name] [column_data_type]

Selecting Information in a SQL Database

Select Specified Column(s)

SELECT [column_name(s)] FROM [table_name]

Select All Columns

SELECT * FROM [table_name]

Select Information If Conditions/Criteria Are Met

SELECT [desired_columns] FROM [table_name] WHERE [condition]

Joining Tables of Information

SELECT [column1_name], [column2_name] FROM [table1_name] INNER JOIN [table2_name] ON [table1_name].[keyfield1] = [table2_name].[foreign_keyfield]
